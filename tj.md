# 마크다운(Markdown) 기본 문법 가이드

이 문서는 마크다운의 자주 사용하는 기본 문법부터 유용한 기능까지 예시와 함께 제공합니다. 아래의 예시들을 직접 확인하며 익혀보세요!

---

## 1. 제목 (Headers)

제목은 `#` 기호를 사용하여 작성합니다. `#`의 개수에 따라 크기가 달라집니다. (1~6단계)

```markdown
# 제목 1 (H1 - 가장 큼)
## 제목 2 (H2)
### 제목 3 (H3)
#### 제목 4 (H4)
##### 제목 5 (H5)   
###### 제목 6 (H6 - 가장 작음)
```

# 제목 1
## 제목 2
### 제목 3

---

## 2. 텍스트 강조 (Emphasis)

글자를 굵게, 기울임꼴, 또는 취소선으로 표시할 수 있습니다.

```markdown
*기울임꼴 (Italic)* 또는 _기울임꼴_
**굵은 글씨 (Bold)** 또는 __굵은 글씨__
***굵고 기울임꼴***
~~취소선 (Strikethrough)~~
```

*기울임꼴 (Italic)*
**굵은 글씨 (Bold)**
***굵고 기울임꼴***
~~취소선 (Strikethrough)~~

---

## 3. 목록 (Lists)

순서가 있는 목록과 순서가 없는 목록을 만들 수 있습니다.

### 순서가 있는 목록 (Ordered List)
숫자와 마침표를 사용합니다. (숫자의 순서가 정확하지 않아도 자동으로 순서가 매겨집니다.)

```markdown
1. 첫 번째 항목
2. 두 번째 항목
3. 세 번째 항목
   1. 하위 항목 1 (들여쓰기 3~4칸)
   2. 하위 항목 2
```

1. 첫 번째 항목
2. 두 번째 항목
3. 세 번째 항목
   1. 하위 항목 1
   2. 하위 항목 2

### 순서가 없는 목록 (Unordered List)
`*`, `+`, `-` 기호를 사용합니다. (주로 `-`를 많이 사용합니다.)

```markdown
- 사과
- 바나나
  - 덜 익은 바나나 (들여쓰기 2칸)
  - 잘 익은 바나나
    - 아주 달콤함 (들여쓰기 4칸)
* 포도
+ 오렌지
```

- 사과
- 바나나
  - 덜 익은 바나나
  - 잘 익은 바나나
    - 아주 달콤함
* 포도

## 4. 링크 (Links)

클릭할 수 있는 하이퍼링크를 만듭니다.

```markdown
[표시할 텍스트](연결할 URL "선택적 툴팁 설명")
[구글](https://google.com "구글로 이동")
자동 링크: <https://naver.com>
```

[구글](https://google.com "구글로 이동")
자동 링크: <https://naver.com>

---

## 5. 이미지 (Images)

링크 문법 앞에 `!`를 붙이면 이미지가 삽입됩니다.

```markdown
![대체 텍스트(Alt Text)](이미지 URL "선택적 툴팁")
![귀여운 고양이](https://placekitten.com/200/150 "야옹")
```

![귀여운 고양이](https://placekitten.com/200/150 "야옹")

*(※ placekitten.com 이미지가 보이지 않을 수 있습니다. 일반적인 이미지 URL을 넣으시면 됩니다.)*

---

## 6. 인용구 (Blockquotes)

다른 글을 인용할 때 사용합니다. `>` 기호를 사용하며 중첩이 가능합니다.

```markdown
> 인용하는 글입니다.
>> 중첩된 인용구입니다.
>>> 더 깊은 인용구입니다.
```

> 인용하는 글입니다.
>> 중첩된 인용구입니다.
>>> 더 깊은 인용구입니다.

---

## 7. 코드 작성 (Code)

문장 중간에 짧은 코드(인라인)를 넣거나, 여러 줄의 코드 블록을 만들 수 있습니다.

### 인라인 코드 (Inline Code)
백틱(`` ` ``) 기호로 감쌉니다.

```markdown
이것은 `console.log('Hello');` 인라인 코드 예시입니다.
```

이것은 `console.log('Hello');` 인라인 코드 예시입니다.

### 코드 블록 (Code Blocks)
백틱 3개(```` ``` ````)로 감싸고 시작 부분에 프로그래밍 언어 이름을 적으면 문법 강조(Syntax Highlighting)가 적용됩니다.

<pre>
```javascript
function greet(name) {
  console.log("Hello, " + name + "!");
}
greet("World");
```
</pre>

```javascript
function greet(name) {
  console.log("Hello, " + name + "!");
}
greet("World");
```

## 8. 수평선 (Horizontal Rules)

구분선을 만들 때 사용합니다. `*`, `-`, `_` 기호를 3개 이상 연속해서 씁니다.

```markdown
---
***
___
```

---

## 9. 표 (Tables)

파이프(`|`)와 하이픈(`-`)을 사용하여 표를 만듭니다. 콜론(`:`)을 사용하여 정렬을 지정할 수 있습니다.

```markdown
| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
| :--- | :---: | ---: |
| 내용 1 | 내용 2 | 내용 3 |
| 짧은 글 | 아주 긴 텍스트 내용 | 짧음 |
```

| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
| :--- | :---: | ---: |
| 내용 1 | 내용 2 | 내용 3 |
| 짧은 글 | 아주 긴 텍스트 내용 | 짧음 |

---

## 10. 체크박스 (Task Lists)

할 일 목록을 만들 때 유용합니다. 목록 기호 뒤에 `[ ]` (빈 칸) 또는 `[x]` (완료)를 넣습니다.

```markdown
- [x] 장보기 (완료)
- [ ] 청소하기 (미완료)
- [ ] 마크다운 공부하기
```

- [x] 장보기 (완료)
- [ ] 청소하기 (미완료)
- [ ] 마크다운 공부하기
